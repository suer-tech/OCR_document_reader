# 📚 Платформа OCR и NLP — База Знаний (LLM Wiki)

Добро пожаловать в центральную базу знаний (LLM Wiki) проекта. Эта Wiki служит единым источником истины (Source of Truth) как для разработчиков, так и для AI-агентов (Antigravity, Codex, Claude Code).

## 🎯 Назначение проекта

Комплексное решение для автоматического распознавания (OCR) и глубокого лингвистического анализа (NLP) судебных решений и других официальных документов. Система позволяет принимать файлы, извлекать текстовые слои, классифицировать типы документов и вытаскивать структурированные сущности (Наименнования организаций, ФИО, ИНН, суммы, номера дел и даты) с высоким качеством.

2. [🤖 Правила для AI-Агентов (AGENT_RULES.md)](/docs/wiki/AGENT_RULES.md)

# Exec‑plans: исполнимые планы для агентов

Каталог `docs/exec-plans/` содержит **исполняемые планы** для крупных изменений.

Каждый файл — это:

- контекст задачи;
- цели и область охвата;
- пошаговый чек‑лист для агентов;
- критерии приёмки.

Исполнимые планы:

- создаются людьми (или по запросу людей);
- обновляются агентами по мере выполнения шагов;
- служат источником правды о том, какие изменения уже внесены и какие ещё активны.

См. общий документ `PLANS.md` для правил работы с планами.

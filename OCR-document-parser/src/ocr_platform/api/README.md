# API

Папка `api/` отвечает только за HTTP контракты.

**Не содержит** бизнес-логику и прямые вызовы БД.

**Главная точка входа**: `main.py` — FastAPI app (функция `create_app()`).

**Схемы**: `schemas.py` — Pydantic модели для валидации запросов/ответов.

# AGENT: При добавлении нового эндпоинта — зарегистрируй его в `main.py` и добавь схему в `schemas.py`.
